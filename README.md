# 略語辞典

社内で使う略語を検索・管理するためのシンプルなWebアプリです。

## 使い方

1. アプリを開く
2. 検索欄にキーワードを入力する
3. 一覧から略語・日本語・英語の意味を確認する
4. 修正したい場合は該当行をクリックして編集する

ローカルで動かす場合:

```bash
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## 新しい略語を追加する

### 方法1: アプリから申請（推奨）

1. 「➕ 新しい略語を追加」をクリック
2. フォームに入力
3. 「🔗 GitHubで申請」をクリック
4. 開いたGitHubページでIssueを送信

### 方法2: ファイルを直接編集

`data/abbreviations.md` に以下の形式で追加します。

```markdown
### API
- **日本語**: アプリケーション・プログラミング・インターフェース
- **English**: Application Programming Interface
- **カテゴリ**: IT
```

## 略語を修正する

### 方法1: アプリから申請

1. 一覧で修正したい略語の行をクリック
2. 内容を修正
3. 「🔗 GitHubで申請」をクリック
4. 開いたGitHubページでIssueを送信

### 方法2: ファイルを直接編集

`data/abbreviations.md` の対象項目を書き換えて、Pull Requestを作成してください。
